# vis.ipynb Guide

The params is colloed in `params.json`. 

You can choose different region by commenting or uncommenting 
For example, 
```Python
# Show all the data
all_region_name = [
    'East Midlands',
    'East of England',
    'London',
    'North East',
    'North West',
    'South East',
    'South West',
    'West Midlands',
    'Yorkshire and The Humber' 
]


# Only show London's data
all_region_name = [
    # 'East Midlands',
    # 'East of England',
    'London',
    # 'North East',
    # 'North West',
    # 'South East',
    # 'South West',
    # 'West Midlands',
    # 'Yorkshire and The Humber' 
]
```
